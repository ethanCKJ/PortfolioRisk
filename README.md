# PortfolioRisk
[![CI](https://github.com/ethanCKJ/PortfolioRisk/actions/workflows/ci.yml/badge.svg)](https://github.com/ethanCKJ/PortfolioRisk/actions/workflows/ci.yml)

## What this does?

This application computes portfolio risk metrics (such as portfolio value, daily P&L, and 95% 1-day historical VaR) on live market data then exposes them via a Grafana dashboard and REST API

## Why did I build it?

This project gave hands on experience in the architecture of large financial applications (event-driven, multi-service, observable) and key industry standard tooling (Docker, Kafka, Grafana, GitHub Actions)
